## 🎯 배포 방법 (3가지 옵션)

### 옵션 1: 자동 배포 도구 사용 (가장 쉬움) ⭐
```bash
# Windows에서 실행
deploy.bat

# 메뉴에서 선택:
# 1. GitHub Pages
# 2. Netlify (드래그 앤 드롭)
# 3. 배포 가이드 보기
# 4. 로컬 테스트
```

### 옵션 2: Netlify 즉시 배포 (2분 소요) ⭐⭐⭐
1. https://app.netlify.com/drop 접속
2. 이 폴더를 드래그 앤 드롭
3. 완료! (자동으로 HTTPS, CDN 제공)

### 옵션 3: GitHub Pages (무료 호스팅) ⭐⭐
```bash
# 1. GitHub에서 새 저장소 생성

# 2. 원격 저장소 연결
git remote add origin https://github.com/YOUR_USERNAME/singer-website.git

# 3. 푸시
git push -u origin master

# 4. GitHub 저장소 Settings > Pages에서 활성화
# 완료! https://YOUR_USERNAME.github.io/singer-website/
```
